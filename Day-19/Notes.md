
## Notes

* Terraform is an Infrastructure as Code (IaC) tool.
* Infrastructure is defined using code instead of manual clicks.
* Terraform is a single executable binary.
* Terraform communicates with cloud providers using their APIs.
* Terraform creates infrastructure; Docker packages applications.
* Infrastructure code can be stored in Git and recreated anytime.
